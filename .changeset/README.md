# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can find the
full documentation for it [in the readme](https://github.com/changesets/changesets/blob/main/packages/changesets-cli/README.md)

The most important thing for you to understand is: a changeset is an intent to release. When you run
`pnpm release:add`, it will ask you which packages have changed and generate a Markdown file in this directory
describing the change. When the Release workflow runs `changeset version`, those files are consumed and the
appropriate `package.json` versions are bumped.
